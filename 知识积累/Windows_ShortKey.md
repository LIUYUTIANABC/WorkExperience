## Windows 常用的快捷鍵

- Win + L           锁屏
- Win + D           显示桌面/恢复功能
- Win + E           快速打开文件资源管理器
- Win + X           快速搜索功能
- Win + +           放大桌面
- Win + V           打开粘贴板
- Win + 左右        改变窗口位置
- Win + Tab         新建桌面
- Win + Ctrl + 左右      切换桌面
- Ctrl + Shift + Esc     打开任务管理器

- Ctrl + O          打开文件

- Alt + 空格 + C  =  Alt + F4     关闭窗口
- Alt + 空格 + N   最小化当前窗口
- Alt + 空格 + X   最大化当前窗口
- Alt + 空格 + R   恢复当前窗口
- Alt + 空格 + M   移动窗口
- Alt + 空格 + S   改变窗口大小
- Alt + 回车       查看文件属性
